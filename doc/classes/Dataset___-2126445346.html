Class: Dataset

Annotations (1)

  • rdfs:comment "This class represents collections of information objects. Datasets can therefore serve as an organizing unit for digital documents especially to make sets of single records identifiable.

    Datasets do not have to be decomposed into single record descriptions. This allows data to be described and retrieved even if it is not mapped on an item level.

    Examples:
    - All records created in the Emagines project
    - All images belonging to the DAI Rome's photo archive
    - The Barrington Atlas roads layer in the geoserver"     @en
