Usage (11)

  • Dataset rdfs:comment "This class represents collections of information objects. Datasets can therefore serve as an organizing unit for digital documents especially to make sets of single records identifiable.

    Datasets do not have to be decomposed into single record descriptions. This allows data to be described and retrieved even if it is not mapped on an item level.

    Examples:
    - All records created in the Emagines project
    - All images belonging to the DAI Rome's photo archive
    - The Barrington Atlas roads layer in the geoserver"     @en 
  • DigitalObject rdfs:comment "All kinds of electronically encoded, machine-readable documents." @en

  • File rdfs:comment "The single unit of a computer file system." @en 
  • Image rdfs:comment "This class comprises distributions of form, tone and colour that may be found on surfaces such as photos, paintings, prints and sculptures or directly on electronic media.

    The same image can be found on different carriers. For example the same photograph can be represented in a negative, a print and several electronic files in different formats.

    Corresponds to E38 Image in the CIDOC-CRM.

    Examples:
    - The photo with number D-DAI-ATH-2014-01000
    - The reconstruction of the Basilica Aemilia created by Florian Willems"     @en
